# spring-ai-demo

基于SpringAI的示例工程，快速体验SpringAI的能力，记录一下个人体验SpringAI进行大模型上层应用开发的学习历程，同时也给希望体验大模型应用开发的java开发者提供一些参考

## 技术栈

- SpringBoot 3.5.3
- SpringAI 1.0.0
- Java17
- Maven
