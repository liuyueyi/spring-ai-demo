# 现代诗创作技能

## 何时使用此技能
- 表达个人情感和内心体验
- 描绘自然景物和生活片段
- 探索哲理和人生思考
- 创作抒情诗、叙事诗、意象诗
- 写城市诗歌、乡村诗歌、爱情诗歌
- 实验性诗歌和创新形式



## Instructions

## 创作指导

### 第一步：理解诗歌主题和情感

**情感表达**：
- 爱情：热烈、温柔、思念、失落
- 孤独：寂静、沉思、内省、疏离
- 自然：敬畏、赞美、融合、感悟
- 生活：温暖、苦涩、希望、迷茫
- 时光：流逝、怀念、珍惜、感叹

**意象选择**：
- 自然意象：风、雨、云、月、花、树、山、河
- 城市意象：霓虹、街道、人群、建筑、车流
- 生活意象：茶杯、书本、窗户、灯光、音乐
- 抽象意象：时间、记忆、梦境、远方

### 第二步：选择诗歌形式和结构

**自由诗**：
```markdown
# 夜晚的思绪

月光洒在窗前，
如银色的纱。
我听见风的声音，
从很远的地方来。

时间在这一刻静止，
只有心跳，
和窗外的虫鸣。
```

**散文诗**：
```markdown
# 城市的黄昏

黄昏时分，整座城市都柔软了下来。霓虹灯次第亮起，像是天空掉落的星星。人群匆匆，每个人都带着自己的故事，在十字路口短暂相遇，然后又各自散去。

我站在天桥上，看着车流如河，忽然明白，我们都是这河流中的一滴水，渺小却真实地流淌着。
```

**意象诗**：
```markdown
# 秋

一片叶子落下
两只鸟飞过
三缕炊烟升起
四野寂静
五更天明

六瓣雪花还未飘落
七色彩虹已挂在心头
八方风雨来时
九月桂花香透了整个秋天
```

**抒情诗**：
```markdown
# 想你

想你的时候
月亮很圆
风很轻
云朵飘得很慢很慢

想你的时候
时间变得很长
像一条没有尽头的路
通向有你的远方

想你的时候
全世界都安静了
只剩下
心跳的声音
```

**叙事诗**：
```markdown
# 老街

那条老街的青石板
被岁月磨得发亮
街角的茶馆里
老人们还在谈论着往事

卖糖葫芦的老人
推着自行车走过
清脆的铃声
唤醒了童年的记忆

如今老街要拆了
那些熟悉的脸庞
那些温暖的故事
都要装进回忆的行囊
```

### 第三步：运用诗歌技巧

**意象营造**：
- 具体可感的形象描写
- 多感官的综合运用（视觉、听觉、嗅觉、触觉）
- 动静结合，虚实相生
- 以小见大，以物寄情

**修辞手法**：
- 比喻：明喻、暗喻、借喻
- 拟人：赋予事物人的特征
- 排比：增强节奏感和气势
- 对比：突出主题和情感
- 通感：跨感官的描写

**节奏韵律**：
- 长短句搭配
- 重复与变化
- 内在的音乐性
- 适当的停顿和留白

**语言特色**：
- 凝练简洁
- 富有张力
- 陌生化表达
- 个性化语言

### 第四步：诗歌示例和赏析

**经典意象运用**：
```markdown
# 海子的《面朝大海，春暖花开》

从明天起，做一个幸福的人
喂马、劈柴、周游世界
从明天起，关心粮食和蔬菜
我有一所房子，面朝大海，春暖花开
```
*赏析：用朴实的语言表达对简单幸福生活的向往*

**现代诗示例**：
```markdown
# 顾城的《一代人》

黑夜给了我黑色的眼睛
我却用它寻找光明
```
*赏析：简短有力，寓意深刻，表达了一代人的精神追求*

**抒情诗示例**：
```markdown
# 徐志摩的《再别康桥》（节选）

轻轻的我走了，
正如我轻轻的来；
我轻轻的招手，
作别西天的云彩。
```
*赏析：轻柔的节奏，淡淡的离愁，意境优美*
### 第五步：润色和完善

**自我检查清单**：
- [ ] 情感真挚，有感染力
- [ ] 意象清晰，富有画面感
- [ ] 语言凝练，无冗余
- [ ] 节奏流畅，有音乐性
- [ ] 立意新颖，有深度
- [ ] 形式与内容统一
- [ ] 留白适当，给读者想象空间

**修改技巧**：
- 删减多余的词语
- 调整句式增强节奏
- 替换平淡的表达
- 强化核心意象
- 优化开头和结尾

### 第六步：添加创作时间

在诗歌完成后，使用 PowerShell 命令获取当前时间，并添加到诗歌末尾：

```powershell
# 获取当前日期和时间
Get-Date -Format "yyyy-MM-dd HH:mm"
```

**示例输出格式**：
```
2026-03-11 10:30
```

或者使用更简洁的命令：
```powershell
date -Format "yyyy-MM-dd HH:mm"
```

**在诗歌末尾添加创作时间**：
```markdown
# 诗题

（诗歌内容）

---
创作于 2026-03-11 10:30
```

或者更文艺的格式：
```markdown
# 诗题

（诗歌内容）

—— 作于 2026 年春夜
```

**注意事项**：
- 时间格式可以根据诗歌风格调整
- 可以使用更诗意的表达（如"作于某年某月"、"写于某个季节"）
- 保持与诗歌整体风格一致
